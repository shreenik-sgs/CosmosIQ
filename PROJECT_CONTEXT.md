# 1. Mission

Build the world's first **Economic Intelligence Operating System (EIOS)**.

The platform is **not**:
- a stock screener
- an AI chatbot
- an equity research tool
- a portfolio optimizer

The platform **is** an operating system for continuously understanding the evolving global economy, discovering new economic knowledge, and converting that knowledge into explainable investment intelligence.
